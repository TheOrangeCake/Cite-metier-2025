# Dependencies
- pygame
- watchdog

# Control
### Preset canva
- `F1` -> first preset
- `F2` -> second preset
- `F3` -> third preset
- `F4` -> fourth preset
- `F5` -> fifth preset
### Input
- `ESC` -> clear input
- `Backspace` -> delete input
- `Enter` -> send input to AI
- `Right shift` -> reset canva to blank
