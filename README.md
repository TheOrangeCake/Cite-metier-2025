# Dependencies
- pygame
- watchdog

# Control
### Preset canvas
- `F1` -> first preset: base game
- `F2` -> second preset: sea scene
- `F3` -> third preset
- `F4` -> fourth preset
- `F5` -> fifth preset
- `F6` -> sixth preset: blank canvas
### Input
- `ESC` -> clear input
- `Backspace` -> delete input
- `Enter` -> send input to AI
