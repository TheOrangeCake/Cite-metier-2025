# Dependencies
- pygame `apt-get install -y python3-pygame`
- watchdog `python3 -m pip install -U watchdog`

# Control
### Preset canvas
- `F1` -> base game
- `F2` -> game sea scene
- `F3` -> game rain scene
- `F4` -> to do
- `F5` -> to do
- `F6` -> blank canvas
- `F7` -> blank city
- `F8` -> blank to do
- `F9` -> blank to do
### Input
- `ESC` -> pause game
- `Backspace` -> delete input
- `Enter` -> send input to AI
